# Causal Relation Extraction Module (Module 1)

A neural network module for the automatic extraction of Causal Relations (Cause and Effect) from unstructured text. It serves as the first core component of an intelligent Decision Support System (DSS).

## Architecture & Key Features

The module is built on the **DistilBERT** transformer architecture using the **BIOES** token-level tagging scheme. It is trained on a high-quality synthetic dataset generated by a Large Language Model (Llama-3) with deterministic marker-to-BIOES conversion.

*   **Performance:** Macro F1 = 0.736
*   **Base Model:** DistilBERT-base-uncased + Linear Classifier
*   **Tagging Scheme:** BIOES (9 classes: O, B/I/E/S-CAUSE, B/I/E/S-EFFECT)
*   **Inference:** Fully local execution, zero external API dependencies.
