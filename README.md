## Overview

This module implements a **neuro-symbolic approach** to causal relation extraction from political texts. It automatically identifies cause-effect relationships from unstructured text using a fine-tuned DistilBERT model trained on LLM-generated synthetic data.

---

## Architecture & Key Features

The module is built on the **DistilBERT** transformer architecture using the **BIOES** token-level tagging scheme. It is trained on a high-quality synthetic dataset generated by a Large Language Model (Llama-3-70B) with deterministic marker-to-BIOES conversion.

- State-of-the-art causal relation extraction (Macro F1: **0.7843**)

- Powered by DistilBERT with BIOES token classification

- Trained on high-quality synthetic data (Llama-3-70B, 1,100 examples)

- Lightweight and efficient (runs on CPU)

- Fully reproducible research code

**Core Specifications:**

-  **Base Model:**  `distilbert-base-uncased` + Linear Classifier

-  **Task:** Token Classification (9 labels: O, B/I/E/S-CAUSE, B/I/E/S-EFFECT)

-  **Tagging Scheme:** BIOES (Begin, Inside, Outside, End, Single)

-  **Dropout:** 0.3

-  **Training:** 6 epochs, batch size 16, lr=2e-5, warmup=10%

-  **Inference:** Fully local execution, zero external API dependencies

## Quick Start

### 1. Download the pre-trained model

Download `best_model.pt` (~254 MB) from [Google Drive](https://drive.google.com/file/d/1W-EmF9BVZzERze2R6LRPVzV7vMUYAe_Q/view?usp=drive_link) and place it in the `models/` folder.

### 2. Run inference

```bash
python  -m  src.predict
```

**Example output:**

```text
Text: The sudden increase in interest rates led to a massive outflow of foreign investment.

Causes: ['The sudden increase in interest rates']

Effects: ['a massive outflow of foreign investment']
```

### 3. Evaluate on test set

```bash
python  -m  src.evaluate
```

### 4. Train from scratch

```bash
python  -m  src.train
```

### 5. Run quick sanity tests

```bash
python  -m  tests.test
```

### 6. Use as API (for Module 2 integration)

```python
from src.predict import CausalPredictor

# Initialize predictor

predictor =  CausalPredictor(model_path='models/best_model.pt')

# Extract causal relations

text =  "Political instability in the region caused a stock market crash."

result = predictor.predict(text)

print(f"Causes: {result['causes']}")

print(f"Effects: {result['effects']}")
```

## Dataset

### Format

Our synthetic dataset uses **BIOES tagging format** (one token per line, tab-separated tag, blocks separated by blank lines):

```text
The O

government O

raised O

taxes B-CAUSE

, O

which O

triggered O

protests E-EFFECT
```

### Files

-  `data/train.txt` - Training set (1,290 examples)

-  `data/val.txt` - Validation set (277 examples)

-  `data/test.txt` - Test set (277 examples)

### Generation

Data was generated using **Llama-3-70B** with marker-to-BIOES conversion. The dataset is balanced between causal and non-causal sentences.
